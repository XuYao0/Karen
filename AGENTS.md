# Project Goal

Build an emotionally supportive companion chatbot with memory.

The long-term objective is to create a chatbot that can provide warm, reliable emotional companionship while remembering important user preferences, personal context, ongoing concerns, and past conversations in a way that improves continuity and trust over time.

Key product principles:

- Emotional companionship should be the core experience, not a secondary feature.
- Memory should make conversations feel continuous, personal, and respectful.
- The chatbot should be safe, privacy-aware, and transparent about what it remembers.
- The system should support gradual improvement from a simple prototype toward a reliable long-term companion.
